Bluff Detection Model using Linear Regression
📌 Project Overview
This project is a simple Machine Learning model developed using Linear Regression to predict salary values based on position levels.

The project demonstrates:
Data loading using Pandas
Data visualization using Matplotlib
Building a Linear Regression model using Scikit-learn
Predicting salary values
Plotting regression results
📂 Dataset Information

The dataset contains:
Position Name
Position Level (Independent Variable)
Salary (Dependent Variable)

Example:
Position	Level	Salary
Business Analyst	1	45000
Manager	4	80000
CEO	10	1000000
